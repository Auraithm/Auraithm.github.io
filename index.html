Wu: A High-Performance and Efficient Diffusion Language Model

Ying Zhu1,2, Jiaxin Wan1,2, Tianyi Liang3,2, Xu Guo1,2, Xiaoran Liu1,2, Zengfeng Huang1,2, Ziwei He2, Xipeng Qiu1,2
1Fudan University, 2Shanghai Innovation Institute,
3East China Normal University
Conference name and year
GitHub Hugging Face Paper

An overview of the Wu framework: an efficient post-training pipeline for diffusion large language models, featuring parallel decoding and advanced optimization techniques.

TL;DR

We release Wu, a high-performance diffusion large language model (DLLM), together with an open-source post-training framework designed for efficient and scalable DLLM adaptation. Our framework enables high-efficiency post-training at generation lengths of up to 8K tokens, substantially reducing computational overhead while maintaining model stability and convergence quality. Leveraging this framework, Wu achieves state-of-the-art (SOTA) performance across a broad range of downstream benchmarks, surpassing existing post-training paradigms in both efficiency and task generalization. This work establishes a practical foundation for advancing long-context DLLM post-training and adaptation.
